This chapter explains the file structure.

## Files

| Filename         | Description                         |
| ---------------- | ----------------------------------- |
| `zely.config.ts` | Configuration File (for typescript) |
| `zely.config.js` | Configuration File (for javascript) |

## Directories

| Directory                     | Description                      | Customizable |
| ----------------------------- | -------------------------------- | ------------ |
| `/pages/`                     | Pages applied to the router      | No           |
| [`/middlewares/`](/docs/auto) | Middleware applied automatically | Yes          |
