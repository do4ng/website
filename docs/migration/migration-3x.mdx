## New loader

The new loader has been introduced in zely. ([Read blog](/blog/25-01-13))

Enable `experimental.useSerpack` to use new loader.

```ts
import { defineConfig } from 'zely';

export default defineConfig({
  experimental: {
    useSerpack: true,
  },
});
```

<Box type="warning" title="Under Development">
  It's not ready to be used for production yet
</Box>
## HTML

No more `ctx.html` like this:

```ts
export default [ALL((ctx) => ctx.html('<h1>Hello World!</h1>'))];
```

Just enable `experimental.useHTML`,

```ts {4-6}
import { defineConfig } from 'zely';

export default defineConfig({
  experimental: {
    useHTML: true,
  },
});
```

and write `.html` file.

```html
<h1>Hello World!</h1>
```
