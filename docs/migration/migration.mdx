## From 1.x to 2.x

`export default` handler no longer provide req and res as parameters. Use [`ctx`](/docs/context) instead.

```ts [title="after"]
import type { PageHandler } from 'zely';

export default [
  GET((ctx) => {
    ctx.send('Hello World!');
  }),
] satisfies PageHandler[];
```

```ts [title="before"]
import type { PageHandler } from 'zely';

export default [
  GET((req, res) => {
    res.send('Hello World!');
  }),
] satisfies PageHandler[];
```
