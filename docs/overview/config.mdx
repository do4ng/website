To apply the config, create `zely.config.js` (or `zely.config.ts`) and then enter the code below to apply it.

<Tabs>
<TabItem id="Typescript">

```ts
import { defineConfig } from 'zely';

export default defineConfig({
  /* options */
});
```

</TabItem>
<TabItem id="Javascript">

```js
module.exports = require('zely').defineConfig({
  /* options */
});
```

</TabItem>
</Tabs>
