In this chapter, we will introduce how to create your first zely app.

## Using tool

You can create zely application easily with [zely-cli](https://npmjs.com/package/zely-cli).

---

1. Download the template using the `zely-cli` tool:

<Tabs>
<TabItem id="Typescript (recommended)">

```bash
npx zely-cli init --dir my-app --template=typescript
```

</TabItem>
<TabItem id="Javascript">

```bash
npx zely-cli init --dir my-app --template=javascript
```

</TabItem>
</Tabs>

2. Install Dependencies:

<Tabs>
<TabItem id="yarn">

```bash
yarn
```

</TabItem>
<TabItem id="npm">

```bash
npm install
```

</TabItem>
</Tabs>

3. Run the app:

<Tabs>
<TabItem id="yarn">

```bash
yarn dev
```

</TabItem>
<TabItem id="npm">

```bash
npm dev
```

</TabItem>
</Tabs>

## Manual Installation

You can also add zely to a project that already exists.

Enter the command below:

```bash [title="npm"]
npm install --save-dev zely
```

Add the configuration file (`zely.config.ts`).

```ts [title="zely.config.ts"]
import { defineConfig } from 'zely';

export default defineConfig({
  // options
});
```
