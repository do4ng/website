## nodeExternal

- **Description**: Excludes `node_modules` from the output.
- **Default**: `false`

<Box type="tip">Refer [Exclude node_modules](/serpack/externals).</Box>
## externals

- **Description**: Specifies modules to exclude from the output.
- **Default**: `[]`

```ts [title=".serpackrc.ts"]
export default <import('serpack').Options>{
  compilerOptions: {
    nodeExternal: false,
    externals: ['serpack'],
  },
};
```

## runtime

- **Description**: Enables runtime functionality, which reduces output size.
- **Default**: `false`

```ts [title=".serpackrc.ts"]
export default <import('serpack').Options>{
  compilerOptions: {
    runtime: true,
  },
};
```

<Box type="tip">Refer [serpack/runtime](/serpack/runtime).</Box>

## parserOptions

- **Description**: Configuration options for the Acorn parser.
- **Default**: `{}`

## globals

- **Description**: Settings for global variables - [`swc.jsc.transform.optimizer.globals`](https://swc.rs/docs/configuration/compilation#jsctransformoptimizerglobals).
- **Default**: _(No default specified)_

## resolverOptions

- **Description**: Options for the `oxc-resolver`.
- **Default**: `{}`

## type

- **Description**: Specifies the type of the script, either `script` or `module`.
- **Default**: `"module"`

## banner

- **Description**: Adds a banner at the top of the output file.
- **Default**: `""`

## footer

- **Description**: Adds a footer at the bottom of the output file.
- **Default**: `""`
