Logger for zely-js

[source](https://github.com/zely-js/zely/tree/zely4/packages/zely-js-logger), [npm](https://www.npmjs.com/package/@zely-js/logger)

## Usage

```js
import { error, warn, success, info } from '@zely-js/logger';

error(new Error('err'));
error('err');

warn('warn');

success('success');

info('info');
```
