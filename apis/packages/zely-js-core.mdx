Core server for zely-js

[source](https://github.com/zely-js/zely/tree/zely4/packages/zely-js-core), [npm](https://www.npmjs.com/package/@zely-js/core)

## Usage

```ts
import { createZelyServer } from '@zely-js/core';

createZelyServer({
  /* options */
}).then((app) => {
  app.listen(8080);
});
```
