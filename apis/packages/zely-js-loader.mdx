Module loader for zely-js

[npm](https://www.npmjs.com/package/@zely-js/loader)

## Usage

```ts
import { createLoader } from '@zely-js/loader';

const loader = await createLoader({
  // custom loaders
  loaders: [],
});
const page = await loader('./index.ts');
```
