`dev` is a command that runs the app in development mode.

```bash
zely-cli dev
```

## Arguments

| **option**  | **description**                   | **default** |
| ----------- | --------------------------------- | ----------- |
| `--serpack` | Use serpack-loader (experimental) | `false`     |
