`init` is a command that helps you easily run the zely app through a template.

```bash
zely-cli init
```

## Arguments

| **option** | **description**                              | **default**  |
| ---------- | -------------------------------------------- | ------------ |
| `dir`      | Dist Directory                               | `.`          |
| `template` | Template Type (`javascript` or `typescript`) | `typescript` |
