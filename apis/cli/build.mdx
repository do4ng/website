---

`build` is a command that builds server.

```bash
zely-cli build
```

<Box type="tip">
  If you want to check the available options and more details, please visit the [API
  page](/apis/build-server).
</Box>

## Bundle

By default, the `build` command does **not** bundle files into a single output to reduce build time.
If you want to bundle everything into a single file, use the `--bundle` flag:

```bash
zely-cli build --bundle
```

## Check Required Modules

To check which modules are required to run the built server, add the `--emit-modules` flag.

<Box type="info">
  This option is provided by the builder itself, not the CLI module. So you need to
  provide the flag in a way that the builder can interpret it correctly. Do **not** add
  extra options like `--emit-modules=true`. Just use the `--emit-modules` flag as is.
</Box>

## Arguments

| **option**       | **description**           | **default** |
| ---------------- | ------------------------- | ----------- |
| `--bundle`       | Bundle into a Single File | `false`     |
| `--emit-modules` | Check Required modules    | `false`     |
