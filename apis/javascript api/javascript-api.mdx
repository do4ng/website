zely provides a rich javascript api.

If you plan to use only the API and not the cli, install `@zely-js/zely` via npm.  
This will help reduce package size. (The `zely` package is a re-export of `@zely-js/zely`, so both can perform the same role.)

```bash
npm i --save-dev @zely-js/zely
```

And load the package.

```ts
import zely from '@zely-js/zely';
```

Now everything is ready!
