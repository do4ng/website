```js [title="zely.config.js"]
module.exports = require('@zely-js/zely').defineConfig({
  /* options */
});
```

## Server Options

- `server.port`: `number` - Server port to use.
- `server.options`: [`zept.ServerOptions`](https://github.com/zely-js/zept/blob/main/packages/http/types/index.d.ts#L33) - HTTP server options.
- `server.zept`: `zept.ZeptServer` - Server instance.

## Zely Options

- `cwd`: `string` - Base directory.
- `middlewares`: `Middleware[]` - [Middlewares](/docs/middleware).
- `allowAutoMiddlewares`: `boolean` - Allow [to import middlewares automatically](/docs/auto).
- `middlewareDirectory`: `string` - Directory including middlewares. [[related](/docs/auto)]

---

- `loaders`: `Loader[]` - List of [loaders](/plugins/create-loader).
- `globalImport`: `boolean` - Whether to import [methods](/docs/methods) automatically.
- `onError`: `Function` - Callback to be called when an error is encountered.
- `enableReporter`: `boolean` - Whether to enable the reporter for debugging. (reporter reports errors exactly through `.js.map`)
- `keepDist`: `boolean` - Whether to keep the dist directory when server starts.
