# Zely documentation

Development documentation based on [exta](https://extajs.netlify.app/)

## Running the website locally

### Prerequisite

- node
- yarn (recommended)

### Installation

```bash
git clone https://github.com/do4ng/website.git
cd website
yarn
```

### Development

```bash
yarn dev
```

### Build

```bash
yarn vite build
```
