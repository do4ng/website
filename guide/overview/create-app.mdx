## 1. Download Template

You can easily download templates using `zely-cli`.

```bash
npx zely-cli init --dir my-app --template=typescript
cd my-app
```

If you set the `--template` option to `typescript`, a typescript template will be downloaded. (default: `javascript`)

## 2. Install Dependencies

Download deps using your preferred package manager.

```bash
npm i
# yarn
# pnpm i
```

## 3. Done!

Now everything is ready.

Go to the next chapter and create a project.
